# NeuroAIBot
Neuro Chat is a high-performance, real-time AI assistant application. It features a seamless, bidirectional communication bridge between a mobile frontend and a Node.js backend, delivering an "instant-reply" experience similar to ChatGPT or Gemini.

**Key Features**
Real-Time Messaging: Powered by Socket.io, messages are delivered instantly with zero refresh delay.
Intelligent AI Engine: Integrated with a free, open-source AI brain (Pollinations) that provides creative and helpful responses to any question.
Premium Visual Experience:
- Custom Brand Identity: Includes a futuristic "Neuro AI" logo and branding.
- Animated Boot Sequence: A high-end splash screen with neural network aesthetics.
- Modern UI: Sleek chat bubbles, distinct user/AI color palettes (Purple/White), and support for selectable text for easy copying.
Production-Ready Backend: A robust Node.js server with built-in automated unit tests to ensure 100% uptime and reliability.

**Technical Stack**
Frontend (Mobile App):
- React Native: Cross-platform mobile development.
- TypeScript: Type-safe code for fewer bugs.
- Safe Area Context: Perfectly optimized for iPhones with notches and modern Android screens.

Backend (Server):
- Node.js & Express: Fast and scalable server architecture.
- Socket.io: Web-socket infrastructure for persistent, real-time data flow.
- Axios: For high-speed API communication with AI engines.

Testing & Quality:
- Jest: Full unit testing coverage for both the app rendering and the server logic.

Project Structure
/RealTimeApp: The mobile application code.
/RealTimeBackend: The intelligent server code.
/assets: High-resolution design assets (icons and splash screens).


Neuro Chat is a complete, full-stack demonstration of modern mobile engineering and AI integration.
